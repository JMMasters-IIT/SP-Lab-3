//This program calculates the count, min, max, average/mean, median, standard deviation, and sum
//of a file that's put into it. 

To run this program unzip the package
Open the command line in Windows and cd to the appropriate folder
Then, run "gcc basicstats.c -o basicstats" without the quotes
Once it's compiled run "basicstats small.txt" or large.txt without the quotes
